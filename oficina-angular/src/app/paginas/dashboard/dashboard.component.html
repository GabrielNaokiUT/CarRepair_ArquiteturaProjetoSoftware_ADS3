Tecnologias utilizadas

Arquitetura monolítica modular com separação obrigatória em camadas. Classes genéricas no pacote core eliminam duplicação de CRUD entre os domínios.

Java 21 LTS Spring Boot PostgreSQL Angular 21 TypeScript Maven Lombok Hibernate JPA
